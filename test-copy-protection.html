복사 방지 기능 테스트

이 페이지는 복사 방지 기능이 활성화되어 있습니다.

테스트할 수 있는 기능들:

  • 텍스트 선택 시도 (드래그)
  • 우클릭 메뉴
  • Ctrl+C (복사)
  • Ctrl+A (전체 선택)
  • Ctrl+X (잘라내기)
  • Ctrl+V (붙여넣기)
  • F12 (개발자 도구)
  • Ctrl+Shift+I (개발자 도구)
  • Ctrl+U (소스 보기)

테스트용 텍스트:

신동탄간호학원은 경기도 화성시에 위치한 전문 교육기관입니다. 간호조무사, 병원코디네이터, 심리상담사 등 다양한 보건의료 관련 자격증 과정을 운영하고 있습니다.

국민내일배움카드 지원으로 부담없이 수강하실 수 있으며, 체계적인 교육과 밀착형 취업 지원을 통해 높은 취업률을 자랑합니다.

관리자 페이지 테스트:

관리자 페이지로 이동 - 관리자 페이지에서는 복사 방지 기능이 비활성화됩니다.

복사 방지 기능 설명:

  • CSS 방어: 텍스트 선택과 드래그 방지
  • JavaScript 방어: 우클릭, 키보드 단축키, 이벤트 차단
  • 관리자 페이지 제외: admin.html에서는 복사 방지 기능이 적용되지 않음
  • 사용자 친화적: 차단 시 알림 메시지 표시
